# causal-transport-tgd

Causal transportability simulation for estimating ketamine treatment effects 
in transgender and gender-diverse (TGD) populations with chronic pain.

Implements a constrained prior predictive Monte Carlo estimator of the 
transported treatment effect ψ, combining a source ketamine cohort with 
target population baseline distributions via Pearl's transportability framework.

- `CompFusion.qmd` — main simulation and sensitivity analyses
